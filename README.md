# 💰 Salary Contribution System – Flask Web Application

A modern and responsive **Salary Contribution System** developed using **Python Flask** in **VS Code**. This web application allows users to enter employee details, calculate salary contribution based on a selected percentage, and instantly view the remaining salary.

## ✨ Features

✔ Employee Name Input
✔ Salary Calculation
✔ Contribution Percentage Input
✔ Automatic Contribution Calculation
✔ Remaining Salary Display
✔ Dark Theme UI with Neon Glow Design
✔ Responsive and Simple Interface

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* Jinja2
* VS Code

## ⚙️ How to Run

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open browser:

```plaintext
http://127.0.0.1:5000/
```
